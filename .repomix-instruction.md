# LangPlug Codebase Instructions

This repomixed codebase represents the complete LangPlug project - a Netflix-style German language learning platform.

## Project Structure

- `src/backend/` - Python FastAPI backend with AI/ML services (transcription, translation, TTS)
- `src/frontend/` - React/TypeScript frontend application
- `scripts/` - Development and deployment scripts
- `.github/workflows/` - CI/CD pipelines

## Key Technologies

- **Backend**: Python 3.11, FastAPI, PyTorch, Transformers
- **Frontend**: React, TypeScript, Vite, TailwindCSS
- **Build System**: Nx monorepo
- **AI Services**: Whisper (transcription), NLLB/Opus-MT (translation), Piper (TTS)

## Development Guidelines

See CLAUDE.md for comprehensive development guidelines including:
- Server management with start-all.bat/stop-all.bat
- Virtual environment usage (api_venv)
- Testing standards and anti-patterns
- PowerShell integration for Windows/WSL environments
